HBV-GLUE

A Sequence Data Resource for Hepatitis B Virus

Hepatitis B virus is a partially double-stranded DNA virus in the genus Orthohepadnavirus, family Hepadnaviridae. The Hepatitis B disease, a chronic infection in hundreds of millions of people worldwide, causes hundreds of thousands of fatalities annually.

HBV-GLUE is a bioinformatics resource for HBV genome sequence data. The web version can be used for basic analysis. An offline version of the resource can be installed by bioinformaticians and used for more advanced work.

Web site highlights

  • A database of HBV sequences and metadata from NCBI, updated daily and arranged into clades (genotypes, subgenotypes).
  • Pre-built, comprehensive multiple-sequence alignments of NCBI sequences, which may be downloaded in user-defined sections.
  • An analysis tool providing phylogenetic genotyping and visualisation of submitted FASTA sequences.

Offline version

The offline version of HBV-GLUE includes all features of the web version plus:

  • A rich set of bioinformatic functions for exploring the dataset and processing HBV sequence data privately.
  • Tools for the analysis of NGS sequences (SAM/BAM files).
